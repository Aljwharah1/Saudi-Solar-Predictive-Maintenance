# ☀️ Sudair Solar — AI Predictive Maintenance System

## 📌 Overview

**Sudair Solar** is an AI-powered predictive maintenance system designed to predict maintenance and cleaning requirements for solar panels operating in desert environments.

The project analyzes environmental and operational sensor data using Machine Learning and engineering-based physical logic to determine when solar panels require maintenance before significant efficiency loss occurs.

The system was inspired by real-world operational challenges faced by large-scale solar power plants in Saudi Arabia, especially desert photovoltaic infrastructures.

---

# 🚀 Problem Statement

Solar panels deployed in desert climates face several operational challenges that directly impact energy production efficiency, including:

- Dust accumulation on solar panels
- High module temperatures
- Reduced energy generation efficiency
- Thermal stress on photovoltaic cells

Traditional maintenance approaches rely on manual inspection and fixed cleaning schedules, which are often inefficient and costly.

This project introduces an intelligent predictive system capable of determining actual maintenance requirements before significant energy production loss occurs.

# 📡 Sensor Infrastructure

The project relies on a monitoring architecture similar to industrial photovoltaic power plant systems.

## Sensors Used

| Sensor Name | Description |
|---|---|
| `IRRADIATION` | Measures incoming solar radiation intensity |
| `AMBIENT_TEMPERATURE` | Measures surrounding environmental temperature |
| `MODULE_TEMPERATURE` | Measures photovoltaic module surface temperature |
| `DC_POWER` | Measures direct current generated by solar panels |
| `AC_POWER` | Measures inverter-converted alternating current output |
| `DAILY_YIELD` | Measures cumulative daily energy generation |

# 🔬 Feature Engineering & Intelligent Classification

The original dataset did not contain direct maintenance classification labels.

A custom engineering methodology was developed based on realistic operational behavior analysis of solar panels using environmental and operational conditions.

## Engineering Logic

The `needs_maintenance` target was generated based on:

- Dust accumulation analysis
- Thermal operating conditions
- Solar irradiation efficiency behavior
- Environmental temperature analysis

A panel is classified as requiring maintenance when:

- Dust accumulation exceeds operational thresholds
- OR module temperature exceeds safe operating limits

This methodology improved:

- Classification realism
- Model reliability
- AI explainability
